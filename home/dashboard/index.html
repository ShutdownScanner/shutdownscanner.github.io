Usage instructions

  • The horizontal axis is time and the vertical axis is the total number of computers on
  • Hover your pointer over the graph to see the data top right
    • Drag a selection to zoom in
    • Double click to zoom out
  • The download button below allows you to save the data
    • The file is in comma separated values (csv) format for use in spreadsheets etc.
    • It includes all recorded data points including ones not present in the graph
